# An Open Nuclear Reactor Simulator

KOMODO DRAGON is a rust translation of [KOMODO](https://github.com/imronuke/KOMODO) which is continuation of [ADPRES](https://github.com/imronuke/ADPRES).

## Features:

- [x] komodo -> main.rs
- [x] mod_data -> data.rs
- [ ] mod_io -> io.rs
- [x] REWRITE
- [x] BACKSPACE
- [x] OPEN
